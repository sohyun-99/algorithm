# 에지 리스트
- 에지를 중심으로 그래프를 표현
- 배열 : 출발 노드 , 도착 노드 저장 + 가중치 저장
   - ex : A[n][2] 의 크기 -> [1,2] 1에서 2로 뻗음
   - ex : A[n][3] 의 크기 -> [1,2,8] 1에서 2로 뻗음 + 8의 가중치
- 벨만 포드 & 크루스칼 알고리즘에 보통 사용

# 인접 행렬
- 2차원 배열 활용

# 인접 리스트 
- ArrayList 활용
- ArrayList[1]에 {2,3}을 연결하는 방식
- 가중치가 있는 경우는 ArrayList[1] = [{2,8},(3,3)]
